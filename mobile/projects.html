Current Projects

Gluon Minecraft

Gluon is the name of a Minecraft modpack put together using the Technic Launcher. I administer the server, both the hardware and software. It is mostly automated with custom bash scripts to handle the start up, shut down and backup of the world files. Associated with this project is the “Let’s Play” video series that is being produced using the server. I also provide technical support for the group.

Currently I am developing a new daemon to administer the Minecraft server with using Python. It is planned to have a web console so that other Moderators can perform maintenance online.
